# 📊 Rossmann Store Sales Forecasting

## 🔍 Problem Statement

Forecast daily sales for Rossmann stores using historical data to support inventory planning and staffing decisions.

## 🎯 Objective

* Predict store-level daily sales
* Capture seasonality, promotions, and store behavior
* Improve forecast accuracy for business planning

## 🛠 Tools & Technologies

* Python (Pandas, NumPy, Scikit-learn, XGBoost)
* Time Series Analysis
* Feature Engineering

## 📈 Key Achievements

* Achieved ~90%+ model accuracy (R² Score)
* Engineered 15+ features including promotions, holidays, and lag variables
* Reduced prediction error significantly using ensemble modeling

## 📂 Files Included

* notebooks/ → EDA & model building
* src/ → reusable pipeline code
* outputs/ → predictions & feature importance

## 🚀 Business Impact

Improved demand forecasting enabling optimized inventory management and reduced stockouts.
